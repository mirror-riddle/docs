Interfaces

Interfaces

Function Types

可以用 interface 来描述函数类型

interface SearchFunc {
  (source: string, subString: string): boolean;
}

Indexable Types

interface StringArray {
  [index: number]: string;
}

可以同时支持number和string两种类型的 index，但是number的 index 对应的类型必须时string的 index 对应类型的子类。因为 javascript 在索引的时候会自动将数字索引转化为字符串索引。

interface {
  [index: string]: string|number;
  [index: number]: number;
}

Class Types

class 可以 implement interfaces

interface ClockInterface {
  currentTime: Date;
  setTime(d: Date): void;
}

class Clock implements ClockInterface {
  currentTime: Date = new Date();
  setTime(d: Date) {
    this.currentTime = d;
  }
  constructor(h: number, m: number) {}
}
